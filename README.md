### 👋 About Me & Work Philosophy

I am a Software Developer specializing in **Web and Mobile App Development**. Through **FE Labs**, I bring ideas to life, focusing on scalable architectures and high-performance user experiences. 

My development philosophy centers around **Agile methodologies** and building lean **MVPs** to test ideas fast. I actively leverage AI coding assistants to optimize my workflow and scale productivity.

- 🚀 **Currently building:** Mobile applications using **React Native (Expo)** and robust backends with **Firebase**.
- 🛠️ **Tech Stack:** From complex web dashboards (React/Next.js) to full-fledged mobile ecosystems.
- 💡 **Workflow:** AI-driven development (Cursor, Claude), rapid prototyping, and seamless deployments.
